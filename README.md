# 🎮 Number Guessing Game

A fun command-line number guessing game built with Python.

## 📌 Project Info
- **Internship:** Synent Technologies – Python Development
- **Task:** Task 2 – Number Guessing Game (Basic Level)

## 🚀 How to Run

**Step 1:** Make sure Python is installed
```bash
python --version
```

**Step 2:** Run the game
```bash
python number_guessing.py
```

## 🎯 Features
- Random number generated between 1–100
- 10 attempts allowed
- Hints: Too High / Too Low
- Attempt counter
- Score rating based on attempts
- Play again option
- Invalid input handling

## 🛠️ Tech Used
- Python 3.x
- `random` module (built-in)
